Trigger

Para que serve e como funciona o Trigger

1

Integridade

Somente categorias que pertençam a mesma classe do item, podem categorizá-lo

2

Como funciona

Trigger funciona ao chamar um procedimento ANTES de executar alguma ação de inserção ou atualização em Categorização

3

Procedimento

Se tipo do item, for diferente do tipo de categoria, levantamos uma exceção no banco de dados.

4

User Interface

Front-end faz requisição para a API que tenta inserir o valor no banco de dados. Caso o trigger retorne erro, a API retornará o erro, informando que o dado não foi inserido.
